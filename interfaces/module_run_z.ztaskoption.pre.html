Interface Pre

Current prerequisite specifier to fill from command line.

Prerequisite specification is started by ZTaskOption.Pre.start method, and concluded either explicitly or by any task specifier modification. Once concluded it is added to the task specifier.

It is an error to modify this specifier before the specification started or after it is concluded.

Available via ZTaskOption.pre property.
