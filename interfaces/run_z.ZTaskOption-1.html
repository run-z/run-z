values

  • values(max?: number): readonly string[]

  • Inherited from ZOption.values

    • Defined in run-z/optionz/index.d.ts:271

    Reads up to the maximum number of values of the option.

    Values are command line options following the currently processed one. Their number depend on the {@link ZOptionSyntax option syntax}. E.g. a --name=value syntax supports one value, while --name value1 value2 ... syntax supports any number of them.

    Calling this method marks the option and value arguments as recognized. This can be undone by calling any recognition method again.

    When called on already recognized option this method just returns the values previously returned to the reader that recognized them.

    Parameters

    • Optional max: number

      The maximum number of values to read.

    Returns readonly string[]

    Option values array.
