applyArgv

  • applyArgv(taskName, argv, opts?): Promise<ZTaskBuilder<TAction>>

  • Recognizes options from process command line argument and applies them to the task.

    This is a method to be called from CLI.

    When the task name specified (typically by npm_lifecycle_event environment variable), finds the corresponding script in package.json, and detects the options specified explicitly. I.e. the ones following the script ones. Then replaces matching command line prefix with the task name and applies the resulting command line options. This makes it possible to specify global options even though the script contains --then or --and option.

    If the above fails, just applies command line options as they are.

    Parameters

    • taskName: undefined | string

      Known task name.

    • argv: readonly string[]

      Command line arguments of the process.

    • Optionalopts: Opts<ZTaskOption, unknown>

      Parser options. By default, starts command line options processing from the third argument.

    Returns Promise<ZTaskBuilder<TAction>>

    A promise resolved to this instance when command line options applied.
